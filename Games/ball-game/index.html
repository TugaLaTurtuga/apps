What is windows destroyer?

Windows destroyer is a program that has a small chance to destroy your windows computer or restart your linux or macOS computer.

How does it work?

It works by deleting most of C:\Windows\System32 including the starting procedure so after restarting the computer it goes to the blue screen.

It is possible to make it work with macOs and linux by deleting /users or /home respectively but it only deletes most of the user's programs if the program is an admin.
